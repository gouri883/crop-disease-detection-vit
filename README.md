# Crop Disease Detection using Vision Transformers

## Overview
This project implements a Vision Transformer (ViT) based deep learning model for automated plant disease classification using leaf images.

## Dataset
- PlantVillage dataset (87,000+ images)

## Model
- Vision Transformer (ViT-B/16)
- Transfer Learning

## Techniques
- Image preprocessing (resize, normalization)
- Data augmentation (rotation, cropping, flipping)
- Fine-tuning using AdamW optimizer

## Results
- Achieved 99.65% validation accuracy
- Compared with CNN baseline (ResNet50)

## Outcome
Demonstrates effectiveness of transformer-based architectures for agricultural image classification.
